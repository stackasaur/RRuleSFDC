# RRuleSFDC

> This library is still very much in progress as of Apr 7, 2023. Feel free to contribute.

This is an implementation of ICalendar Recurrence Rule rule. For more information, [see the specification here](https://icalendar.org/iCalendar-RFC-5545/3-8-5-3-recurrence-rule.html)
